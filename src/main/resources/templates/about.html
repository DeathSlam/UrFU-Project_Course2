О приложении

Веб-приложение «Список дней рождений и подарков» - это удобный инструмент для планирования и учёта подарков к дням рождения.

Возможности приложения:

  • Создание и управление списком дней рождений
  • Добавление подарков к каждому дню рождения
  • Автоматический расчёт общей и средней стоимости подарков
  • Система ролей и прав доступа
  • Разграничение доступа к данным между пользователями

Технологии:

  • Spring Boot - основной фреймворк
  • Spring Security - аутентификация и авторизация
  • Spring Data JPA - работа с базой данных
  • Thymeleaf - шаблонизатор
  • H2 Database - встроенная база данных
  • Hibernate - ORM
  • Lombok - упрощение кода

Роли пользователей:

  • READ_ONLY - только просмотр собственных записей
  • USER - создание и управление собственными записями
  • ADMIN - полный доступ ко всем данным и управление ролями
